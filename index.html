My Orienteering Programs (open source)

GPX or TCX track to map A single html file to align a .gpx or .tcx track to a (scanned) map (.png). The track can have or not have any controls (laps from a Garmin GPS watch). You just click on the track or the controls and drag the circles to the correct place on the map. Just aligning two points is often enough for a gross alignment. To download press "d" key.

Maps overview Orienteering maps overview. Python script that scans all the .jpg maps (with EXIF GPS data, exported from QuickRoute) and generate a topographic overview map of them. The generated maps.html file should be opened with a web browser.

Tracking

QuickRoute A Windows program to analyze GPS tracks from orienteering or running. You can load a map and a GPS track and see your route on the map. You can also see speed, heart rate and altitude graphs synchronized with the track on the map. Free to use but not open source.
